
# Hostel Management System

## Overview
A comprehensive hostel management application designed to streamline operations including room allocation, student records,complaints, and facility management.

## Features
- **Room Management**: Track room availability and assignments
- **Student Records**: Maintain detailed student information and documentation
- **User Authentication**: Separate logins and role-based dashboards for students and administrators
- **Complaint Management**: Log and resolve maintenance issues
- **Reports**: Generate occupancy and financial reports

## Tech Stack
- Frontend: [React, TailwindCSS, JavaScript]
- Backend: [Node.js, Express.js]
- Database: [PostgreSQL]

## Installation
```bash
git clone https://github.com/chaitanyaraj75/Hostel-Management
cd "Hostel Management"
```
For backend
```bash
cd "backend"
npm install
node server.js
```
For frontend
```bash
cd "frontend"
npm install
npm run dev
```
